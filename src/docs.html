Base rules

Base rules are the defaults. They are almost exclusively single element selectors but it could include attribute selectors, pseudo-class selectors, child selectors or sibling selectors.

This whole page demonstrates default rules applied on html like background, color, font-family, font-size, font-weight etc.

SASS variables

Inside sass/ folder you will find vars.scss that holds SASS variables with Myworkout design details.

Available variables:

Font utilities
$mw-brand-font-family - Default font family
$mw-default-font-size - Default font size
$mw-default-font-weight - Default font weight
$mw-default-line-height - Default text line height
Colour palette
  
$mw-color-brand-1
  
$mw-color-brand-2
  
$mw-color-neutral-1
  
$mw-color-neutral-2
  
$mw-color-neutral-3
  
$mw-color-neutral-4
  
$mw-color-neutral-5
  
$mw-color-neutral-6
  
$mw-color-positive-feedback
  
$mw-color-negative-feedback
Brand gradient
  
$mw-gradient-brand-1 (expression)
selector { background: $mw-gradient-brand-1; }
  
$mw-gradient-brand-2 (expression)
selector { background: $mw-gradient-brand-2; }
