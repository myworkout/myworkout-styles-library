Base rules

Base rules are the defaults. They are almost exclusively single element selectors but it could include attribute selectors, pseudo-class selectors, child selectors or sibling selectors.

Base rules for Myworkout profile's global scope are defined in src/sass/base.scss file.

Typography

Typography rules specified in this library are related to the scope of an article or .mw-article. However it is possible to use these styles outside of article context, by using provided classes.

.mw-text - Default text style
.mw-title - Default style for a page title. Inside article or .mw-article this is default style for h1.
.mw-subtitle - Default style for subtitles. Inside article or .mw-article this is default style for h2.
.mw-heading - Default style for third level headings. Inside article or .mw-article this is default style for h3.
.mw-lead - Default style for intro paragraph.
.mw-p - Default style for a paragraph. Inside article or .mw-article this is default style for p.
.mw-quote - Default style for quotes. Inside article or .mw-article this is default style for blockquote.

SASS variables and constants

Inside sass/ folder you will find constants.scss and variables.scss that hold Myworkout design details. Constant values should not be overridden by client apps. Variables can be overridden if and when appropriate.

Available constants:

Font utilities
$MW_BRAND_FONT_FAMILY - Default font family
$MW_DEFAULT_FONT_SIZE - Default font size
$MW_DEFAULT_FONT_WEIGHT - Default font weight
$MW_DEFAULT_LINE_HEIGHT - Default text line height
