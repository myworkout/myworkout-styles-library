SASS variables

Inside sass/ folder you will find vars.scss that holds SASS variables with Myworkout design details.

Available variables:

Font utilities
$mw-brand-font-family - Default font family
$mw-default-font-size - Default font size
$mw-default-font-weight - Default font weight
Colour palette

$mw-color-brand-2
